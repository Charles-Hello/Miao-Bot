# Miao-Bot
一个想用go来玩bot的辣鸡选手
这个文件夹gin-vue-admin-main是以后bot的前端框架，还没写。https://github.com/flipped-aurora/gin-vue-admin想玩的看作者部署即可。

目前已实现的功能：
支持微信机器人http交互
支持对话式交流，
支持指定普通任务开关，
支持正则，管理员，判断，快速编写的一个小go插件脚本。
